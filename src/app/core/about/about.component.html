About PRS

A better way to manage purchase requests.

What is the PRS product?

The Purchase Request System (PRS) is the capstone for the MAXimum boot camp and its completion is required for graduation. It's made up of a database, a backend, and a frontend using various technologies. The database contains five table: User, Vendor, Product, Request, and Requestline.

The backend implements Create, Read, Update, and Delete (CRUD), allowing end-users to have full control over data stored in the database.

Frontend consist of a login page, home page, about page, and components that allow transactions for each of the database tables.
